2018-02-18

数据结构

线性表

数组

  • n个数据的优先序列
  • 大小固定，一旦创建大小就无法改变
  • 当前数组不能再存储新元素的时候，创建一个大数组替换当前数组，从而实现动态数据结构
  • 可以通过下标访问、修改元素，效率高，插入和删除的花费比较大

链表

  • 由多个内存中不相邻的节点连接而成，每一个节点包括了data-数据内容，next-指向的下一个节点

树与二叉树

树

  • n个有限节点组成的集合
  • 具有层次关系
  • 每个节点有0-n个子节点
  • 有0个或者1个父节点
  • 如果没有父节点，则是根节点

二叉树

  • 树的一种
  • 每个节点最多只有两个子节点
  • 便利方法：
    • 每一个的三个基本单元：根节点，左子树，右子树
    • 先序遍历：
      • 若二叉树为空，则空操作
      • 否则县访问根节点，再先序遍历左子树，然后先序遍历右子树
    • 中序遍历：
      • 若二叉树为空，则空操作
      • 否则先中序遍历左子树，然后访问根节点，然后中序遍历右子树
    • 后序遍历
      • 若二叉树为空，则空操作
      • 否则先后序遍历左子树，然后后序遍历右子树，然后访问根节点
  • 二叉查找树
    • 左子树上所有节点的值均小于根节点
    • 右子树上所有节点的值均大于他的根节点
    • 没有两个节点的值相等
  • 平衡二叉树
    • AVL树，左子树右子树都是平衡二叉树
    • 左子树右子树深度之差的绝对值不超过1
  • 红黑树
