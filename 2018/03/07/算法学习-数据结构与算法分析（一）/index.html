2018-03-07

序言

第一章主要讲的是C++的编程基础，曾经学过但已经遗忘差不多，此外英文的实在看着头痛，固参考了中文版C++ Primer进行略读

各种库

iostream

  • 输入输出流
  • cin>>
  • cout<<
  • cerr 错误信息
  • clog 一般信息

各种数据类型

bool，char，wchar_t，short，int，long，float，double，long double

各种函数

main函数

  • main函数是唯一被操作系统显示调用的函数
  • 一般返回0-成功，其他整数-失败

各种类与相关内容

  • interface & implementation

其他高级货

  • 引用：&，int &quote = val， quote是val的别名 const引用是对于const的引用
  • typedef用来给type取名字
  • enum，枚举 enum val={a=1,b=2,c=3}
  • vector frac(10)
  • 迭代器iterator 容器的begin end
